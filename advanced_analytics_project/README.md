# sql_advanced_analytics_project

A comprehensive collection of SQL scripts for data analytics, and reporting. These scripts cover various analyses such as time-based trends, cumulative analytics, data segmentation, performance analysis,part to whole comparisons. This repository contains SQL queries designed to help data analysts and BI professionals quickly explore, segment, and analyze data within a relational database. The script focuses on a specific analytical theme and demonstrates best practices for SQL queries.
The report consolidates key customer metrics and behaviors.
