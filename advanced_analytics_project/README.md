# advanced_analytics_project

### 📖 Project Overview:

This repository contains advanced SQL analytics performed on the Gold Layer of the Data Warehouse project. It builds upon the curated analytical model to generate deeper business insights using advanced SQL techniques such as window functions, aggregations, and analytical reporting.

This script covers various analyses such as time-based trends, cumulative analytics, data segmentation, performance analysis,part to whole comparisons. This file contains SQL queries designed to help data analysts and BI professionals quickly explore, segment, and analyze data within a relational database. The script focuses on a specific analytical theme and demonstrates best practices for SQL queries.  
  
The report _customers consolidates key customer metrics and behaviors.
